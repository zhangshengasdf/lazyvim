# 练习 1 参考答案 — Neo-tree 文件操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 新建文件 `utils/helper.lua` | `a` | 输入 `utils/helper.lua`，自动创建中间目录 |
| 删除 `temp.txt` | `d` | 选中文件后按 d，确认后删除 |
| 重命名为 `new-name.lua` | `r` | 选中文件后按 r，输入新名称 |
| 复制到 `backup/` | `c` → 移动到目标目录 → `p` | 先复制，再粘贴 |
| 显示/隐藏 gitignore 文件 | `I` | 大写 I，切换 gitignore 文件的显示 |
